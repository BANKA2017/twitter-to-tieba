# twitter-to-tieba
把指定用户的推特转发到贴吧<br />
现已支持翻译(不完美),转推(支持转任何推特用户的推到任何贴吧的任何帖子里面),自动写入log,无需数据库(数据库版待开发),检查cookie完整性<br />
来自[KDWNIL开发者](https://kdwnil.ml)<br />
demo：获取方式：GET<br />地址：https://kdwnil.ml/dev/twitter/?name=被搬运者的twitterid&tid=贴的id&bdn=贴所在贴吧的名字

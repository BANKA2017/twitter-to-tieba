# twitter-to-tieba
把指定用户的推特转发到贴吧<br />
现已支持翻译(不完美),转推(支持转任何推特用户的推到任何贴吧的任何帖子里面),自动写入log,无需数据库(数据库版待开发),检查cookie完整性<br />
来自[KDWNIL开发者](https://kdwnil.ml)<br />
[简易文档](https://kdwnil.ml/dev/twtotb/getmytwtotb.html)<br />
